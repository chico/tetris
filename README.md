# tetris

## Setup arduino

* Connect joystick to the Arduino

![arduino](https://brainy-bits.com/wp-content/uploads/2014/12/schematic-joystick1.png)

## Get code

* ```git clone git@github.com:chico/tetris.git```
* ```cd tetris```

## Install and run

* ```npm install```
* ```node index```
